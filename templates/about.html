My Companion is a next-generation web-based AI assistant designed to feel more like a digital friend than a chatbot. It combines natural language understanding, emotional awareness, and memory-based interactions to create meaningful conversations.

This project demonstrates how Artificial Intelligence, Data Science, and Web Development can work together to build an intelligent, responsive, and human-centered system.

The frontend is built using HTML, CSS, and JavaScript, while the backend is powered by Python (Flask) and integrates with the OpenAI API to generate real-time, intelligent responses.
