About My Companion

Runtime overview

My Companion is a Flask-based AI cockpit that blends conversational replies, lightweight emotion awareness, persistent user memory, and a more cinematic interface layer.

Backend intelligence

LIBRA supports Gemini-powered replies when an API key is configured, then drops into a built-in offline mode so the experience keeps moving.

Memory + behavior

Conversation memory, preferences, and behavior signals can use Firebase when available, and automatically fall back to a local JSON store when needed.

Interface layer

The frontend combines server-rendered templates, theme switching, hover depth, live telemetry, and motion tuned for desktop and mobile.

Design direction

More command center, less plain dashboard.

The current version leans into atmospheric lighting, bold display typography, glass panels, and interaction cues that feel intentional.

Current stack

  • Flask
  • Google Gemini
  • Firebase
  • HTML, CSS, JavaScript
  • Render or Gunicorn deployment
